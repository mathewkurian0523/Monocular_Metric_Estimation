# Monocular Metric Estimation

A computer vision pipeline for estimating **real-world scale from a single RGB image** using object detection and geometric reasoning.

This project demonstrates how **reference objects detected in an image can be used to infer approximate real-world measurements** from monocular images.

---

## Overview

Estimating real-world size from a single image is challenging because depth information is not directly available.

This project uses **object detection and known reference object dimensions** to estimate metric scale.


## Applications

Robotics perception

Augmented reality measurement tools

Scene understanding systems

Industrial inspection

Autonomous systems
